# E-Commerce Platform Database System

## Project Overview
This project is a relational database system for an e-commerce platform built using SQL. It manages customers, suppliers, categories, products, orders, payments, shipping, and reviews.

## Features
- Structured relational schema
- Primary keys and foreign keys
- Constraints and normalization
- Data insertion, update, and deletion
- Filtering, sorting, joins, and aggregations
- Stored procedures
- Trigger for logging orders
- Transaction-based payment processing
- Indexing for query optimization
